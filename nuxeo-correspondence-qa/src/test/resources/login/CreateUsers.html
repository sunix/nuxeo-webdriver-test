Login: Create Users

This is creating the users that will be used in the other tests

  • Having that
    • The user Administrator is the default one (unchanged)
    • The server is started, connecting to http://localhost:8080/nuxeo displays the default login page.
  • Login with administration rights to create Users
    • When
      • the user fills the following informations: Administrator with the password Administrator and the language English (United States)
      • the user login
    • Then
      • the user is connected
  • Create user John Doe
    • When
      • With a username jdoe, a firstname John , a lastname Doe , a company Nuxeo , a passwordjdoe1, a email jdoe@nuxeo.com and in a group members,
      • the administrator creates a new user
    • Then
      • the user created is present in the list of users.
