Working with filing trees

Classification trees are folders associated to the mailboxes in which users can classify case items. It enables them to find and organize documents another way. Classification trees work the same way folders do in Nuxeo DM. Access is determined by access rights, documents can be deleted and moved into the folder's trash.

Access rights are inherited from the mailbox. This means that all the mailbox delegates automatically have access to the mailbox's classification folders.

You can classify cases as soon as:

  • you can access them (ie when you are a delegate of a recipient mailbox)
  • and you have "Classify" right in the target classification folder.

The complete functional documentation can be found here: Working with filing trees
