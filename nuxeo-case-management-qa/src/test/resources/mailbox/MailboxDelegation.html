Mail: Mailbox Delegation

Mailboxes have delegates which are users that have access to it. In Correspondence (and case management) there is no right access as DM has: if a user is a delegate of a mailbox, it has full access to it. Administrators can assign and remove users or groups of users as delegates of the mailbox.The complete functional documentation can be found here: Manage a mailbox
