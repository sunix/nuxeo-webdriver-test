Nuxeo Case Management server start instruction

Nuxeo case management server startup instruction.

  • Start the server if its not started
    • The property case.management.distribution.location is set. it represents the location of the distribution zip file used by our concordion specs
    • The server should be installed in nuxeo-cm-tomcat
    • Use the command
      unzip nuxeo-case-management-distribution-*-tomcat-cmf.zip
      unzip the server if needed
    • The command
      chmod +x nuxeo-cm-tomcat/bin/nuxeoctl
      gives execution permission to the server binaries
    • The command
      nuxeo-cm-tomcat/bin/nuxeoctl start
      starts the server
